Gestiona la MetaInformacion de objetos java utilizando Introspeccion.

Introspeccion

Obtiene la información sobre los objetos, clases, propiedades, etc,... utilizando el mecanismo de Introspeccion propio de java. (las clases deben estar en el ClassPath de la Maquina Virtual Java que está corriendo).

JRPropertyDescriptor

Es un mecanismo para ampliar la información de la introspección normal de Java. Lo usamos en el modeler, pero creo que no se usa en las aplicaciones en producción. Consiste en crear un clase BeanInfo (vease manual de java), y utilizar JRPropertyDescritor en lugar PropertyDescriptor.
