Utilidades para manejar la estructura de los objetos (metainformacion). Nos da la información sobre las clases (identificadores, nombres, etc,..) y los atributos que tienen (nombres, tipos, editores,...)
Las interfaces fundamentales son:
  • MetaInfoFactory:Devuelve la información sobre una clase.
  • MetaEntity:Contiene la información de una clase
  • MetaAttribute:Contiene la información de los atributos

Entidades

Aunque normalmente, los objetos manejados son una instancia de Java, y se manejan por 'introspeccion' es posible crear elementos a los que asignamos un tipo, que puede no existir como clase java. Esos elementos los llamamos Entitys.

Entidades Dinamicas

Son las implementaciones de Entity, cuando no existe la clase del tipo indicado, y 'nos inventamos' sus propiedades, o sus instancias

Intro

Es el paquete que implementa la metainformación para clases java que ya estan creadas. Para ello utiliza la introspección.

xml

Es la implementación de una 'factoria' de información que está almacenada en un fichero XML.

creators

Es una utilidad para crear datos aleatorios (o secuenciales)

Editores

Cada propiedad tiene un 'editor' asociado. El editor es un String que representa la forma de tratar esa propiedad. Es responsabilidad de la implementación del runner instrumentar correctamente esos editores.
