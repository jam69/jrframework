Modelo de Formularios

Formularios

Contiene la descripción de formularios. Los formularios están formados por componentes (item) que se encuentra en celdas (cell). Los items tienen propiedades y son de tipos variados. Es responsabilidad de la implementación de los runners, el conocer los tipos de items y de manejar su implementación. (en el Web hay una plantilla por cada item, en el swing hay una clase con un mapa de nombres-clases,...)
Actualmente solo tenemos un layout para formularios. Está heredado del FormLayout (jgoodies.FormLayout) para swing. Define para el panel una serie de filas y de columnas, y luego va asignando los items a sus celdas. En el futuro se harán otros tipos de Layout.
